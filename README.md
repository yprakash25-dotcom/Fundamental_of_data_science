# Fundamentals of Data Science Assignments

This folder contains short Python programs for basic programming practice (input, output, conditions, loops, and math operations).

**Requirements**
- Python 3.x

**How To Run**
Use a terminal in this folder and run any file, for example:
```bash
python Q2operations.py
```
# Introduction
Python is one of the most high-level and user-friendly free programming languages, mainly applied in data science, artificial intelligence, and software development. It facilitates the execution of highly complex calculations by a developer with syntax that is very easy to write. This assignment is focused on developing foundational programming skills in Python using the following:
• Input and output operations
• Arithmetic calculations
• Conditional statements
• Loops and logical problem-solving
• Basic algorithm design
Most of these tasks simulate hands-on experiences with practical problem-solving approaches, which are absolute prerequisites for data science.

**Files**
- `Q2operations.py` Basic arithmetic operations on two numbers.
- `Q3classification.py` Classifies a number as positive/negative and even/odd, or zero.
- `Q4math.py` Calculates cube, cube root, logs, and power of 6 for a number.
- `Q5marks.py` Reads 6 subject marks and computes total, average, percentage, and grade.
- `Q6primes.py` Lists prime numbers in a range and shows count and sum.
- `Q7divisible.py` Lists numbers divisible by 9 but not by 6 within a range.
- `Q8factorial.py` Computes factorial of a positive integer with input validation.
- `Q9menu.py` Menu program to sum positive and negative numbers separately.
- `Q10game.py` Number guessing game with limited attempts.
- `flowchartpython.png` Flowchart image for the assignment.
